* Custom libraries for Java projects

## Install

* Add a dependency in your POM.

	```xml
	<dependency>
		<groupId>com.github.spt-oss</groupId>
		<artifactId>spt-java-commons</artifactId>
		<version>X.X.X</version>
	</dependency>
	```
